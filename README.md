# Weather Multi‑Agent System


Проект демонстрирует многошаговый LLM‑пайплайн с погодными агентами и интеграцией с OpenWeather API.


## Функционал
- Получение погоды по городу
- Интерпретация погоды агентом
- Выбор вечернего досуга
- Подбор еды для пикника с учётом ограничений
- Рекомендации мест в городе
- Рекомендации по одежде
